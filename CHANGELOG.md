# Changelog

All notable changes to `madr-lint` are recorded here. Releases are managed via [changesets](https://github.com/changesets/changesets); each entry below corresponds to a published npm version.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) and the project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html) starting at v1.0.0. Pre-1.0 releases (`0.x`) treat minor bumps as potentially breaking.

## Unreleased

The first published release will be `0.1.0-alpha.0`. See `git log` for the milestone history (M0 scaffold → M5 production dogfood).
